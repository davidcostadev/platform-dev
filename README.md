# GraphQL APIs
http://localhost:3000/graphql

## Teams
### all  :[Team]
```graphql
query{
	team{
		 id, name,members,link
	}
}
```

### read by id :[Team]
```graphql
query{
	team(args:{id:12}){
		 id, name, contacts
	}
}
```

### validate Edit Key :Boolean
```graphql
query{
	teamEditKey(args:{id:<ID> ,key:<EDIT_KEY>})
}
```


### validate View Key :Boolean
```graphql
query{
	teamVIEWKey(args:{id:<ID> ,key:<VIEW_KEY>})
}
```
